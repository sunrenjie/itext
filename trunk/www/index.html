Project description
iText is a library that allows you to generate PDF files on the fly.

The iText classes are very useful for people who need to generate read-only, platform independent documents containing text, lists, tables and images. The library is especially useful in combination with Java(TM) technology-based Servlets: The look and feel of HTML is browser dependant; with iText and PDF you can control exactly how your servlet's output will look.
iText requires JDK 1.2. It's available for free under the LGPLicense.
Why I wrote these classes
I worked on a project for Ghent University and I had to write an application that could be used by all the University professors of Ghent to enter the grades of their students into the student database. The application also had to calculate the final results of the students and performed a first (automated) deliberation at the end of the academy year.

A lot of documents had to be generated:
  • Lists for the professors to fill in the grades of every student after an exam.
  • Advalvas lists with the results of an exam.
  • Lists with all the theses of each graduation year of each faculty.
  • Pre- and post-deliberation lists for the different deliberation commissions of every faculty.
  • Reports with the curriculum of each student, his (or her) grades and the global grade he (or she) obtained.
  • ECTS-reports
  • and so on...
This application was built as an internetsite. It is entirely written in JAVA(TM), using JAVA(TM) technology-based Servlets to generate HTML pages and PDF documents.

This JAVA-PDF library is a part of this application. The code of this library was published under the GNU LIBRARY GENERAL PUBLIC LICENSE.
