This list of features is somewhat out of date, but it can be a good starting point to learn to work with iText. Every topic is a link to the (outdated, but still very useful) iText tutorial. You will find examples that are more up-to-date on Paulo's website.

A new tutorial will be written soon (Nov 11th - Nov 20th 2004).
