Part III: Advanced PDF

Chapter 10: Graphics
Creating the Graphic object
To draw lines and curves, you need to create a Graphic object. On this object, you can perform actions such as: rectangle(...), moveTo(...), lineTo(...),... Check out the class PdfContent or the API for a description of all the possible methods. In the API, I refer to the 1st edition of the PDF Reference Manual version 1.3 (PDFSPEC.PDF).
In example 1, you can see how a rectangle and its diagonal are drawn: see the resulting pdf.
