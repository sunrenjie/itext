Part II: Other document formats

Chapter 8: RTF
RtfWriter
The RTF package in iText is a very experimental package that is not a part of the core iText library. The code is in separate tar.gz, zip and jar-file together with the also experimental XML functionality. This package was written by Mark Hall. We (Bruno Lowagie and Paulo Soares) don't know a lot about this package, but we thought it could be useful for some of you, so we added it to the library.
We made some very simple examples that show some of the RTF possibilities:
  • Example 1 has some paragraphs and an anchor
  • Example 2 has a simple list
  • Example 3 has a simple table
That's all we can say about RTF functionality for now. We kind of hope some of you can help with further development of the RTF functionality.
