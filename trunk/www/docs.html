Installation
This library doesn't need to be compiled or installed. Just be sure that the iText037.jar-file is added to your CLASSPATH or that the complete classfile tree (com/lowagie/text/*) is in a directory to which the CLASSPATH refers.
Javadoc
You can consult the API online.
Samples
You can find the source of two standalone examples here: Text.java and Tables.java.
There is also a Servlet showcase on the following address: http://webappcabaret.com/blowagie/itext/
