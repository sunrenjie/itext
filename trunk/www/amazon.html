Overview of some Adobe Products

Adobe Acrobat Standard 6.0
(windows version) (Mac version)
Upgrade Windows Upgrade Mac
  • Easily convert any paper or electronic document to Adobe PDF
  • Capture research information from the Web and combine it with reports
  • Makes it easy to track, manage, and incorporate feedback into original documents
  • Batch process PDF files with settings for adding security
  • Control whether users are allowed to view, edit, comment on, sign, or print PDF files
Adobe PageMaker 7.0
(windows version) (Mac version)
  • Create professional-quality publications
  • Build from scratch or choose from hundreds of templates
  • Achieve a uniform look with master frames to hold text and graphics
  • Professional typesetting with kerning and tracking
  • Includes thousands of clip art photos and illustrations
FrameMaker 7.0
(windows version) (Mac version)
  • Full support for XML authoring and publishing
  • Support for tagged Adobe PDF (Portable Document Format)
  • Enhanced HTML publishing capabilities
  • SVG (Scalable Vector Graphics) support
  • Starter applications, starter templates, and an XML cookbook

Adobe Acrobat Professional 6.0
(windows version) (Mac version)
Upgrade Windows Upgrade Mac
  • Create PDF files with the click of a button from Microsoft applications and AutoCAD
  • Convert and combine application files into one compact PDF file in a single step
  • Create lists of reviewers and track the feedback you receive
  • Create interactive electronic forms that can be completed and submitted online
  • View two areas of a large drawing simultaneously by using a split-pane view
Adobe Illustrator 10.0
(windows version) (Mac version)
  • Create vector-based graphics for the Web, emerging media, and print
  • Keep file sizes small and manage repeating graphics with new symbolism tools
  • Bend, stretch, and twist design elements and retain editability
  • Generate custom artwork variations using dynamic data-driven graphics
  • Optimize Web graphics with object-based slicing and cascading style sheet layers
Acrobat Capture 3.0
(windows version)
  • Convert scanned pages to searchable Adobe PDF files that you can view, navigate, and print
  • Create reusable document-processing workflows tailored to different types of conversion projects
  • Accurately perform OCR, font, and page recognition
  • Automatically create intradocument links, including tables of contents, cross references, and indexes
  • Use the new Zone tool to define areas of scanned pages as images, text, or even keywords
