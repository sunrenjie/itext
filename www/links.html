Products These are some products that use the iText library:
  • 2STEP: 2step is a template system for developing web applications programmed in Java.
  • PdfDoclet: PDFDoclet is a javadoc doclet which produces PDF output for a given Java API
  • JPedal: The library for extracting data from Pdf files in Java
  • JFreeReport: JFreeReport is a Java class library for generating reports. It has the following features: full on-screen print preview; data obtained via the Swing TableModel interface (making it easy to print data directly from your application); XML-based report definitions; output to the screen, printer or Acrobat PDF format;
  • JasperReports: JasperReports is a powerful report-generating tool that has the ability to deliver rich content onto the screen, to the printer or into XML and PDF files. (LGPL!)
  • HelpExpert: the tool to create electronic documentation!
  • iFrame, a Visual Framework for Cold Fusion: this is an open source presentation layer for database-driven web applications that uses iText
  • Jaydoh: Jaydoh(TM) is a modular development platform which enables the rapid development of web and wireless applications through a collection of reusable and customizable components. Jaydoh uses open standards and is completely platform and datasource independent.
  • RReport Visual Builder: design your reports visually with this tool.
  • art0: art0 pretends to be an artistical world simulator.
